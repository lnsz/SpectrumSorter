# Spectrum Sorter

A way to visualize sorting algorithms by using a colour spectrum

## Usage

Compile solution file with Visual Studio
Go to the output folder (./x64/Debug/)
Run from command line:
  ./SpectrumSorter.exe [sorting_algorithm]
